## 🎯 VISION
**Prototype d'appariement sémantique basé sur des embeddings vectoriels.**

Une recherche explorant l'utilisation du NLP pour résoudre l'asymétrie d'information dans le recrutement. Le système vectorise les compétences et les offres pour calculer un score de proximité sémantique au-delà du simple mot-clé.
